# X (Twitter) Clone

A responsive X (Twitter) UI clone built using React.js and Tailwind CSS while learning MERN development from CodeWithHarry's Sigma Web Development Course.

## About The Project

This project was created mainly to practice and improve Tailwind CSS skills, component structuring, responsive layouts, and modern UI design in React.

The goal was not to build a perfect production-level clone, but to understand how real-world social media layouts are designed and structured.

Due to time limitations, some features and responsiveness are simplified.

## Features

- X/Twitter inspired UI
- Dark theme layout
- Responsive sidebar behavior
- Sticky navigation sections
- Hover animations
- Feed layout
- News/Trending sidebar
- Tailwind CSS styling
- React component structure

## Tech Stack

- React.js
- Tailwind CSS
- JavaScript
- Vite

## Learning Purpose

This project was built during the learning phase of:
- Tailwind CSS
- React component architecture
- Responsive design basics
- Modern frontend UI development
